# Password_locker_app

This is a terminal run application that lets a user create an account where they could save their social medias accounts details.


## Technologies Used

- Python3.8


##### Setup Instructions and Installation

- Clone the repository to a local folder. Ensure you have Python3.8 installed in your machine. 
- Open the folder in your prefered editor. 
- Open the terminal, run the application by executing ./run.py  in the terminal.
